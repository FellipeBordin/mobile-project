# 🚗 Vehicle Project

Sistema completo para gestão financeira de veículos, permitindo controle de despesas, vendas e análise de lucro/prejuízo.

O projeto possui duas aplicações:

- 🌐 (Next.js)
- 📱 Mobile (React Native com Expo)

---

## 🛠 Tecnologias Utilizadas

### 🌐 (vehicle-Next)
- Next.js
- React
- TypeScript
- Prisma ORM
- PostgreSQL
- Tailwind CSS

### 📱 Mobile (vehicle-mobile-app)
- React Native
- Expo
- TypeScript
- Expo Router

---

## 📂 Estrutura do Projeto

vehicle-Next/ → Api
vehicle-mobile-app/ → Aplicação Mobile


---

## ✨ Funcionalidades

- Cadastro de veículos
- Registro de despesas
- Venda de veículo
- Cálculo automático de lucro/prejuízo
- Visualização detalhada por veículo
- Navegação por abas (Mobile)
